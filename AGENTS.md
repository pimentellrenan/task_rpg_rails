# AGENTS.md

Task RPG Rails app. Personal organization app for fast task capture, mobile use, iPhone Shortcuts, and light gamification.

Rules:
- Keep this repo separate from `nextjs_space`.
- Local first. Do not wire production, Vercel, Supabase, or GitHub remote unless explicitly requested.
- Prefer Rails conventions, Hotwire, Stimulus, Tailwind, PostgreSQL.
- UI must be mobile-first for iPhone 16 browser use.
- Portuguese-first copy.
- Docker is source of truth for local runtime.
